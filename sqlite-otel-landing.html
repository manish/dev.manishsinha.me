Quick Start

# Download and run
curl -L https://github.com/manish/sqlite-otel/releases/latest/download/sqlite-otel-linux-amd64 -o sqlite-otel
chmod +x sqlite-otel
./sqlite-otel --otlp-grpc=4317 --otlp-http=4318

That's it! Your collector is now ready to receive telemetry data.
